Sente-se de frente para o aparelho com o peito contra o encosto do banco e pegue os pegadores com os braços estendidos ao nível dos ombros. 

Deite-se em um banco plano com o peito contra o banco, pegue os halteres com as palmas para dentro (uma virada para a outra) os braços estendidos para baixo na linha dos ombros. Levante os halteres para cima até o nível das orelhas no arco mais distante possível, mantendo os cotovelos ligeiramente dobrados. Abaixe os halteres de volta à posição inicial.
